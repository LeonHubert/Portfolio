Over mij

Ik ben Leon, een student van 21 jaar oud met een passie voor programmeren en ontwerpen. Ik studeer momenteel Communication en multimedia design aan de Haagse Hoge School in Den Haag. Mijn sterke kant is mijn drive om programmeerproblemen te tackelen met efficiënte oplossingen. Mijn doel is om mijzelf verder te ontwikkelen qua programmeerkennis in HTML, CSS en JS. Om dit doel te bereiken wil ik eigen projecten ondernemen zoals een eigen forum bijhouden of zelf een game runnen. Ik ga een aansluitende stage zoeken in een bedrijf die mij kan helpen deze doelen te verwezenlijken. Momenteel heb ik een bijbaantje bij PicNic om wat centen te pakken, dit is een bedrijf waar klantvriendelijkheid en stiptheid de hoogste prioriteit hebben, dit is dan ook iets waar ik voor sta!

Wat kan ik

Ik beheers de basis van HTML en CSS. Ik heb meer ervaring met JS aangezien ik sinds dat ik 13 ben bezig ben met programmeren. Ik heb gewerkt met Java, C++, C#, VB, PAWN en Javascript. Ik ben momenteel werkzaam als developer voor een gaming server en community met meer dan 60000 unieke geregistreede spelers. Ik ontwerp maps voor de server, ik programmeer de maps met bijbehoorende code, test en implementeer deze.

Waar ik aan gewerkt heb

Ik heb voor een schoolopdracht een verbetering van een webshop gemaakt, ik heb bij deze opdracht mijn kennis over patterns ingezet.

Waar ik ook aan gewerkt heb

Ik beheers de basis van Adobe Photoshop 2018 en van Adobe Illutrator 2018. Ik heb hierin een poster ontworpen voor een fictief evenement in de toekomst genaamd Robotica.

Mijn meest recente werk

Ik werk momenteel als developer voor www.gta-dayz.com. De server heeft meer dan 60000 unieke spelers in de database. Ik programmeer in de programmeertaal PAWN, dit is een bron-gelimiteerde variant op C. Ik beheers deze programmeertaal volledig sinds 2019.

Waar ik voor sta

Ik omschrijf mijzelf als een efficiënte programmeur en designer. Ik presteer goed onder druk. Ik streef naar efficiëntie, doelgerichtheid en effectiviteit in mijn programmatuur en ontwerpen.
